# Tabs

## Install

In the target component add:

```
import {ATabGroup, ATab, ATabHeading} from "@cisco-ats/atomic-react/ATabs";
// or if you're able to treekshake node_modules
import {ATabGroup, ATab, ATabHeading} from "@cisco-ats/atomic-react";
```

## Router Integrations

On `ATab`, set the `component` property to your router's link type, for example:

```
// With react-router:
<ATab component={RouterLink} to="/">Tab 1</ATab>
```

## Accessibility Notes

By default, `ATabGroup` components are assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [tabslist](https://www.w3.org/TR/wai-aria/#tabslist).

By default, `ATab` components are assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [tab](https://www.w3.org/TR/wai-aria/#tab).

The `ATab` `click` event handler is triggered both by the `click` event and by the `keyDown` event for the `Enter` key.

Each `ATab` component is tab-focusable with a visual overlay.
