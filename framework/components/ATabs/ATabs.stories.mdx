# Tabs

## Install

The component is available on [Cisco's Artifactory package registry](https://devhub.cisco.com/artifactory/webapp/#/home).

```
  npm i --save @cisco-ats/atomic-react
```

And in the target component add:

```
import {ATabGroup, ATab, ATabHeading} from "@cisco-ats/atomic-react/ATabs";
```
