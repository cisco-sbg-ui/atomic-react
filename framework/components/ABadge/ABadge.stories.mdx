# Badges

## Install

Integrate with your build to [auto-import](?path=/docs/getting-started--page#integrating), or add an import in your component:

```
import ABadge from "@cisco-ats/atomic-react/ABadge";
// or if you're able to treekshake node_modules
import {ABadge} from "@cisco-ats/atomic-react";
```

## Accessibility Notes

By default, `ABadge` components are assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [alert](https://www.w3.org/TR/wai-aria/#alert) which denotes that the alert "is a live region with important and usually time-sensitive, information."
