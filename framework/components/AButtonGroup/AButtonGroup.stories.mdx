# Button Groups

## Install

In the target component add:

```
import AButtonGroup from "@cisco-ats/atomic-react/AButtonGroup";
// or if you're able to treekshake node_modules
import {AButtonGroup} from "@cisco-ats/atomic-react";
```

## Usage

<Preview>
  <Story name="usage-1">
    <div style={{marginRight: 20}}>
      <h3>Button Group - Multiple</h3>
      <YourComponent>
        {() => {
          const [selectedValues, setSelectedValues] = useState([
            "One",
            "Three"
          ]);
          return (
            <>
              <AButtonGroup
                multiple
                selectedValues={selectedValues}
                onChange={(_, selectedValues) =>
                  setSelectedValues(selectedValues)
                }>
                <AButton value="One">One</AButton>
                <AButton value="Two">Two</AButton>
                <AButton value="Three">Three</AButton>
              </AButtonGroup>
              <ADivider />
              Selected Values: {JSON.stringify(selectedValues)}
            </>
          );
        }}
      </YourComponent>
    </div>
