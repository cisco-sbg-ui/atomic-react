# Lists

## Install

In the target component add:

```
import {
  AList,
  AListItem,
  AListItemAvatar,
  AListItemSubtitle,
  AListItemTitle,
  AListItemContent,
  AListItemAction
} from "@cisco-ats/atomic-react/AList";
// or if you're able to treekshake node_modules
import {
  AList,
  AListItem,
  AListItemAvatar,
  AListItemSubtitle,
  AListItemTitle,
  AListItemContent,
  AListItemAction
} from "@cisco-ats/atomic-react";
```

## Router Integrations

On `AListItem`, set the `component` property to your router's link type, for example:

```
// With react-router:
<AListItem component={RouterLink} to="/">List Item 1</AListItem>
```

## Accessibility Notes

The `AListItem` `click` event handler is triggered both by the `click` event and by the `keyDown` event for the `Enter` key.
