# Contextual Notifications

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import AContextualNotification from "@cisco-ats/atomic-react/AContextualNotification";
// or if you're able to treekshake node_modules
import {AContextualNotification} from "@cisco-ats/atomic-react";
```
