# Panels

## Install

In the target component add:

```
import {
  APanel,
  APanelHeader,
  APanelFooter,
  APanelBody,
  APanelTitle
} from "@cisco-ats/atomic-react/APanel";
// or if you're able to treekshake node_modules
import {
  APanel,
  APanelHeader,
  APanelFooter,
  APanelBody,
  APanelTitle
} from "@cisco-ats/atomic-react";
```
