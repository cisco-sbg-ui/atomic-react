# Panels

## Install

In the target component add:

```
import {
  APanel,
  APanelHeader,
  APanelFooter,
  APanelBody,
  APanelTitle
} from "@cisco-ats/atomic-react/APanel";
```

## Usage

<Preview>
  <AApp>
    <APanel>
      <APanelHeader>
        <APanelTitle>
          Feature: Panels
        </APanelTitle>
        <AIcon>settings</AIcon>
      </APanelHeader>
      <APanelBody>
        Panels create a sense of discreteness in the content.
      </APanelBody>
      <APanelFooter>
        Use panels today!
      </APanelFooter>
    </APanel>
  </AApp>
</Preview>
