# Buttons

## Install

In the target component add:

```
import AButton from "@cisco-ats/atomic-react/AButton";
// or if you're able to treekshake node_modules
import {AButton} from "@cisco-ats/atomic-react";
```

## Usage

<Preview>
  <Story name="usage-1">
    <AApp>
      <YourComponent>
        {() => {
          const [count, setCount] = useState(0);
          return (
            <div>
              <AButton onClick={() => setCount(count + 1)}>Click Me</AButton>
              <div>Button has been clicked {count} times.</div>
            </div>
          );
        }}
      </YourComponent>
    </AApp>
  </Story>
</Preview>

## Link Buttons

If the `href` property is used, the component will render as an HTML anchor.

<Preview>
  <AApp>
    <AButton secondary href="https://www.cisco.com">
      Launch Cisco.com
    </AButton>
  </AApp>
</Preview>

## Router Integrations

Set the `component` property to your router's link type, for example:

```
// With react-router:
<AButton component={RouterLink} to="/">Button 1</AButton>
```

## Animations

Animations are enabled by default on the `AApp` component and can be disabled.

<Preview>
  <AApp>
    <AButton>Animated</AButton>
  </AApp>
  <AApp animations={false}>
    <AButton>Not Animated</AButton>
  </AApp>
</Preview>
