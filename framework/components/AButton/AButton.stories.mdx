# Buttons

## Install

The component is available on [Cisco's Artifactory package registry](https://devhub.cisco.com/artifactory/webapp/#/home).

```
  npm i --save @cisco-ats/atomic-react
```

And in the target component add:

```
import AButton from "@cisco-ats/atomic-react/AButton";
```

## Usage

<Preview>
  <YourComponent>
{() => {
  const [count, setCount] = useState(0);
  return (
    <div className="atomic-ui-root--animated">
      <AButton primary onClick={() => setCount(count + 1)}>
        Click Me
      </AButton>
      <div>
        Button has been clicked {count} times.
      </div>
    </div>
  );
}}
  </YourComponent>
  <div>
    <AButton secondary to="https://www.cisco.com" target="_blank">Launch Cisco.com</AButton>
  </div>
</Preview>
