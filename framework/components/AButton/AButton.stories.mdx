# Buttons

## Install

In the target component add:

```
import AButton from "@cisco-ats/atomic-react/AButton";
```

## Usage

<Preview>
  <Story name="1">
    <AApp>
      <YourComponent>
    {() => {
      const [count, setCount] = useState(0);
      return (
        <div>
          <AButton
            className="fern"
            onClick={() => setCount(count + 1)}>
            Click Me
          </AButton>
          <div>
            Button has been clicked {count} times.
          </div>
        </div>
      );
    }}
      </YourComponent>
    </AApp>
  </Story>
</Preview>

## Link Buttons

If the `href` property is used, the component will render as an HTML anchor.

<Preview>
  <AApp>
    <AButton secondary href="https://www.cisco.com">
      Launch Cisco.com
    </AButton>
  </AApp>
</Preview>


## Animations

Animations are enabled by default on the `AApp` component and can be disabled.

<Preview>
  <AApp>
    <AButton>
      Animated
    </AButton>
  </AApp>
  <AApp animations={false}>
    <AButton>
      Not Animated
    </AButton>
  </AApp>
</Preview>
