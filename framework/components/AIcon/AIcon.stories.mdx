# Icons

## Install

In the target component add:

```
  import AIcon from "@cisco-ats/atomic-react/AIcon"
```

## Usage

<Preview>
  <AApp>
    <AIcon className="status-yellow--text">warning</AIcon>
    <AIcon size={32} className="status-red--text">
      block
    </AIcon>
    <AIcon size="large">add</AIcon>
  </AApp>
</Preview>

## Accessibility Notes

By default, `AIcon` components use the [aria-label](https://www.w3.org/TR/wai-aria/#aria-label) attribute to label the icon by name. This can be overridden by setting the `label` prop on the component.
