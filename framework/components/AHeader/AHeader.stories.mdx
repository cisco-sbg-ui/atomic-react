# Headers

## Install

In the target component add:

```
import {
  AHeader,
  AHeaderLogo,
  AHeaderTitle,
  AHeaderNavigation
} from "@cisco-ats/atomic-react/AHeader";
// or if you're able to treekshake node_modules
import {
  AHeader,
  AHeaderLogo,
  AHeaderTitle,
  AHeaderNavigation
} from "@cisco-ats/atomic-react";
```

## Accessibility Notes

By default, `AHeader` is assigned the [TR](https://www.w3.org/TR/html51/) tag of [header](https://www.w3.org/TR/html51/sections.html#elementdef-header) which denotes that it represents introductory content for its nearest ancestor sectioning content or sectioning root element. A header typically contains a group of introductory or navigational aids.
