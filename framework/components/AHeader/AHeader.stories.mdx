# Headers

## Install

In the target component add:

```
import {
  AHeader,
  AHeaderLogo,
  AHeaderTitle,
  AHeaderNavigation,
  AHeaderToolbar
} from "@cisco-ats/atomic-react/AHeader";
```

## Accessibility Notes

By default, `AHeader` is assigned the [TR](https://www.w3.org/TR/html51/) tag of [header](https://www.w3.org/TR/html51/sections.html#elementdef-header) which denotes that it represents introductory content for its nearest ancestor sectioning content or sectioning root element. A header typically contains a group of introductory or navigational aids.

If the `AHeaderToolbarItem` has either a `href` or `onClick` handler specified, it is made tab-focusable with visual overlay.
If the `AHeaderToolbarItem` has an `onClick` handler specified, it is assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [button](https://www.w3.org/TR/wai-aria/#button).
