# Timeline

## Install

Integrate with your build to [auto-import](?path=/docs/getting-started--page#integrating), or add an import in your component:

```
import {
  ATimeline,
  ATimelineItem,
  ATimelineItemBody,
  ATimelineItemTitle
} from "@cisco-ats/atomic-react/ATimeline";

// or if you're able to treekshake node_modules
import {
  ATimeline,
  ATimelineItem,
  ATimelineItemBody,
  ATimelineItemTitle
} from "@cisco-ats/atomic-react";
```
