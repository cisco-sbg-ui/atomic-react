# Toasts

## Install

Integrate with your build to [auto-import](?path=/docs/getting-started--page#integrating), or add an import in your component:

```
import {useAToaster} from "@cisco-ats/atomic-react/AToaster";
// or if you're able to treekshake node_modules
import {useAToaster} from "@cisco-ats/atomic-react";
```

## Usage

## Accessibility Notes

The toaster applies the [aria-live](https://www.w3.org/TR/wai-aria/#aria-live) attribute with a value of `polite` to alert the user of the new content.

Also applied is the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [status](https://www.w3.org/TR/wai-aria/#status) to indice it is a `live region`.
