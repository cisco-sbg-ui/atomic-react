## Color Palette

Apply color overrides using helper classes:

- To override the background, add `className="[color_name]--[color_variant]"`, for example: `className="fern--wash-1"`.
- To override the text color, add `className="[color_name]--[color_variant]--text"`, for example: `className="dark-blue--burnt-1--text"`
- For either background or text color classes, there is no need to specify the base color variant. For example, `className="cisco-blue white--text"`

## Helper classes

`AApp` includes a number of helper functions with breakpoint options.

They generally take the form `.[property]-[breakpoint/print]-[value]`, where specifying the breakpoint is optional. Targeting print media is also an option.

For example, `d-flex d-md-block d-print-none`:

- Displays as a flexbox from a 0 to 991 pixel screen width.
- Displays as a block from 992 pixels and higher.
- Does not display when printed.

### Breakpoints

Breakpoints are set to the following:

- xs: 479px
- sm: 767px
- md: 992px
- lg: 1200px
- xl: 1440px

### Overflow

- `.overflow-[overflow-value]`: Example: `overflow-auto`
- `.overflow-x-[overflow-value]`: Example: `overflow-x-auto overflow-x-xs-scroll`
- `.overflow-y-[overflow-value]`: Example: `overflow-y-scroll overflow-y-lg-auto`

### Display

- `.d-[display-value]`: Example: `d-flex d-sm-block d-print-none`

### Float

- `.float-[float-value]`: Example: `float-left float-xl-none`

### Flex

- `.flex-[flex-direction-value]`
- `.flex-[flex-wrap-value]`
- `.flex-fill`: overrides the flex property to `1 1 auto`.
- `.flex-[(grow|shrink)-(0|1)]`: set the flex grow/shrink values to zero or one. Example: `flex-shrink-0 flex-xl-shrink-1`
- `.justify-[justify-content-value]`
- `.align-[align-items-value]`
- `.align-content-[align-content-value]`
- `.align-self-[align-self-value]`
- `.order-[order-value]`: includes `order-first`, `order-last` in addition to `order-(0-12)`

### Margins

- `.ma-[0-12|auto]`: adds margins to `all` sides
- `.mx-[0-12|auto]`: adds margins to left and right sides
- `.my-[0-12|auto]`: adds margins to top and bottom sides
- `.mt-[0-12|auto]`: adds margins to top
- `.mr-[0-12|auto]`: adds margins to right
- `.mb-[0-12|auto]`: adds margins to bottom
- `.ml-[0-12|auto]`: adds margins to left

### Padding

- `.pa-[0-12|auto]`: adds padding to `all` sides
- `.px-[0-12|auto]`: adds padding to left and right sides
- `.py-[0-12|auto]`: adds padding to top and bottom sides
- `.pt-[0-12|auto]`: adds padding to top
- `.pr-[0-12|auto]`: adds padding to right
- `.pb-[0-12|auto]`: adds padding to bottom
- `.pl-[0-12|auto]`: adds padding to left

### Text

- `.text-[text-align-value]`
- `.text-[white-space-value]`
- `.text-break`: breaks lines of text mid-word
- `.text-truncate`: to have text trailing off in an ellipsis.
- `.text-[text-transform-value]`
