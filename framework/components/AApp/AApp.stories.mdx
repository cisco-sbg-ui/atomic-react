## Color Palette

Apply color overrides using helper classes:

- To override the background, add `className="[color_name]--[color_variant]"`, for example: `className="fern--wash-1"`.
- To override the text color, add `className="[color_name]--[color_variant]--text"`, for example: `className="dark-blue--burnt-1--text"`
- For either background or text color classes, there is no need to specify the base color variant. For example, `className="cisco-blue white--text"`

## Helper classes

`AApp` includes a number of helper functions with breakpoint options.

They generally take the form `.[property]-[breakpoint/print]-[value]`, where specifying the breakpoint is optional. Targeting print media is also an option.

For example, `d-flex d-md-block d-print-none`:

- Displays as a flexbox from a 0 to 991 pixel screen width.
- Displays as a block from 992 pixels and higher.
- Does not display when printed.

### Media Queries and Breakpoints

Media queries are set for the following breakpoints:

- xs: 479px
- sm: 767px
- md: 992px
- lg: 1200px
- xl: 1440px
- print (for print styles)

### Reference

- `overflow`
  - not responsive
  - values: `auto`, `hidden`, `scroll`, `visible`
  - examples: `overflow-visible`
- `overflow-x`
  - not responsive
  - values: `auto`, `hidden`, `scroll`, `visible`
  - examples: `overflow-x-hidden`
- `overflow-y`
  - not responsive
  - values: `auto`, `hidden`, `scroll`, `visible`
  - examples: `overflow-y-scroll`
- `display`
  - prefix: `d`,
  - is responsive
  - values: `none`, `inline`, `inline-block`, `block`, `table`, `table-row`, `table-cell`, `flex`, `inline-flex`
  - examples: `d-flex d-print-none`
- `float`
  - is responsive
  - values: `none`, `left`, `right`
  - examples: `float-left float-xl-none`
- `flex`
  - is responsive (not print)
  - values: `fill`
  - examples: `flex-fill, flex-sm-fill`
  - notes: sets the styling to `flex: 1 1 auto`
- `flex-direction`
  - prefix: `flex`
  - is responsive (not print)
  - values: `row`, `column`, `row-reverse`, `column-reverse`
  - examples: `flex-row-reverse flex-sm-row`
- `flex-grow`
  - prefix: `flex`
  - is responsive (not print)
  - values: `grow-0`, `grow-1`
  - examples: `flex-grow-0 flex-lg-grow-1`
- `flex-shrink`
  - prefix: `flex`
  - is responsive (not print)
  - values: `shrink-0`, `shrink-1`
  - examples: `flex-shrink-1 flex-xs-shrink-0`
- `flex-wrap`
  - prefix: `flex`
  - is responsive (not print)
  - values: `wrap`, `nowrap`, `wrap-reverse`
  - examples: `flex-wrap flex-md-nowrap`
- `justify-content`
