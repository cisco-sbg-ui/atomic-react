# Date Picker

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {ADatePicker} from "@cisco-sbg-ui/atomic-react";
```

## Usage

<Playground
  code={`() => {
  const [date, setDate] = useState(new Date(2021, 0, 14));
  return (
    <ADatePicker value={date} onChange={(date) => setDate(date)} />
  );
}
`}
/>

## Range

<Playground
  code={`() => {
  const { value, onChange, } = useADateRange();
  return (
    <ADatePicker value={value} onChange={onChange} />
  );
}
`}
/>

### Initial Range Date

<Playground
  code={`() => {
  const { value, onChange, } = useADateRange([new Date(), new Date(2022, 3, 5)]);
  return (
    <ADatePicker value={value} onChange={onChange} />
  );
}
`}
/>
