# Button Cards

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import AButtonCard from "@cisco-ats/atomic-react/AButtonCard";
// or if you're able to treekshake node_modules
import {AButtonCard, AButtonCardTitle, AButtonCardBody} from "@cisco-ats/atomic-react";
```

## Usage

<Playground
  code={`() => {
  const [count, setCount] = useState(0);
  return (
    <div>
      <AButtonCard onClick={() => setCount(count + 1)}>
      <AButtonCardTitle>Click Me</AButtonCardTitle>
       <AButtonCardBody>Button has been clicked {count} times</AButtonCardBody>
</AButtonCard>
    </div>
  );
}
`}
/>

```

<AButtonCard component={RouterLink} to="/">Title</AButtonCard>
```
