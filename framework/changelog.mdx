## 3.0.1 (2021-08-05)

**BREAKING CHANGES**:

- Switched package hosting from Cisco's Artifactory to GitHub Packages
- Package is now a public package
- The `ADropdown` component is now obsolete and removed. Use `AMenu`. `ASelect`, `ACombobox`, and `AAutocomplete` now solely use `AMenu` internally, and their `useMenu` prop is obsoleted
- Per-component imports (ex: `import ADivider from "@cisco-sbg-ui/atomic-react/ADivider"`) have been removed. Use `import {ADivider} from "@cisco-sbg-ui/atomic-react"`
- Fonts are now hosted from `https://brand-assets.security.cisco.com/`
- `--state-selected` modifier has been standardized to `--selected`

**Other changes**:

- Updated styles to match atomic 3.0.0
- Added `appendContent` and `prependContent` to `AAutocomplete`, `ACombobox`, and `ASelect` to allow for content to be prepended/appended to the menu items
- Added `itemTemplate` to `AAutocomplete`, `ACombobox`, and `ASelect` to allow for custom menu items
- Added Cisco Secure colors
- Added new icon `diverge`
- Fixed a null `ref` issue with `ATabGroup`
- Fixed an `ASelect` race condition triggered by opening and closing the menu in less than 50 milliseconds
- Fixed `AListItem` `role` prop to display custom value when in `AMenu`
- Fixed scrollbar corner color
- Docs: updated README, including a section on contributions
- Docs: updated `Getting Started`
- Docs: fixed initial state of `ASelect` usage example
